# Diabetes Prediction with Machine Learning

Ce projet a pour objectif de prédire la présence du diabète chez un patient à partir de données médicales en utilisant des techniques de Machine Learning.

Le projet comprend les étapes suivantes :

- Exploration et analyse des données (EDA)
- Nettoyage et préparation des données
- Visualisation des variables importantes
- Division des données en ensembles d'entraînement et de test
- Entraînement de modèles de classification
- Évaluation des performances des modèles
- Comparaison des résultats obtenus

## Technologies utilisées

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset

Le projet utilise le dataset **Pima Indians Diabetes Database**, contenant plusieurs caractéristiques médicales telles que :

- Nombre de grossesses
- Taux de glucose
- Pression artérielle
- Épaisseur du pli cutané
- Niveau d'insuline
- Indice de masse corporelle (BMI)
- Fonction héréditaire du diabète
- Âge

## Objectif

Développer un modèle capable de prédire si un patient est susceptible d'être atteint du diabète à partir de ses données médicales.

## Résultats

Les performances des modèles sont évaluées à l'aide de métriques telles que :

- Accuracy
- Precision
- Recall
- F1-Score
- Matrice de confusion
